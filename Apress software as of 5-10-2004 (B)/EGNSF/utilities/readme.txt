utilities

These Shared and stateless classes expose a number of utility methods 
for development tasks common to Windows and Web applications. For its
Issues and its Change Record, see the flagship class utilities.VB and
commonRegularExpressions.VB.

This project/DLL contains two files:


     *  The Shared (static) utilities.VB class contains a variety 
        of handy methods for string handling, math and other common tasks
        including low-intensity parsing and integer arithmetic in 
        multiple number systems.

     *  commonRegularExpressions.VB is an internal Friendly class
        that defines common regular expressions for the 
        Public commonRegularExpressions method in utilities.VB.
